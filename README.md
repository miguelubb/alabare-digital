# Alabaré Digital - Versión Distribución

Esta es la versión lista para distribuir del sitio web Alabaré Multimedia, optimizada para navegadores embebidos en iPad, Android y otros dispositivos.

## Contenido

- `index.html` - Página principal
- `css/style.css` - Estilos
- `js/app.js` - Lógica de la aplicación
- `js/songs_data.js` - Datos de canciones con letras embebidas
- `Musica_Alabare_II/` - Archivos MP3 de Alabaré II
- `Musica_Alabare_III/` - Archivos MP3 de Alabaré III
- `alabareI/` - Archivos MP3 de Alabaré I

## Cómo usar

1. Abre `index.html` en cualquier navegador web moderno
2. Selecciona una categoría en el menú lateral
3. Haz clic en una canción para ver su letra y reproducir el audio
4. Usa el buscador para encontrar canciones por nombre o ID

## Características

- ✅ Interfaz moderna con Bootstrap 5
- ✅ Reproductor HTML5 (sin Flash)
- ✅ Letras con acordes alineados
- ✅ Búsqueda en tiempo real
- ✅ Categorías temáticas
- ✅ Funciona sin conexión a internet (después de la primera carga)
- ✅ Responsive para dispositivos móviles y tablets

## Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- JavaScript habilitado
